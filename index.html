Consejos básicos de viaje

Viaja ligero

Si es posible, lleva solo equipaje de mano. Es más cómodo y casi siempre suficiente para escapadas cortas.

Mapas fuera de línea

Descarga mapas en tu móvil antes de salir. Te servirán incluso sin cobertura.

Seguro de viaje

Comprueba que tienes cobertura médica adecuada, sobre todo si viajas fuera de Europa.

Sobre este blog

Esta página se ha creado como ejercicio para practicar desarrollo web básico en el lado del cliente.

Todo el código se ha escrito a mano utilizando HTML para la estructura, CSS para los estilos y JavaScript para añadir interactividad sencilla.

Tecnologías utilizadas

  • HTML5 para la estructura semántica.
  • CSS3 para el diseño y la maquetación.
  • JavaScript para actualizar el contenido sin recargar.
  • GitHub Pages para el despliegue estático.
